Watch My Workout

Mobile, Smartwatch

WatchMyWorkout is an Android phone application with an Android Wear smartwatch companion app. The smartwatch app is used to track a user's activity, such as running or weight lifting. The data is saved on the smartwatch until it is connected to the phone app. The main focus of the phone app is to show the user his or her progress over a period of time, thus motivating them to continue exercising. The phone app also allows a user to set an exercise schedule. By completing this project I gained more knowledge about Android development and Android Wear interfaces and the sensors available to the watch.

Appli

Web, Mobile

Appli is a mobile and web app that I did with my classmate Paul Conroy last year for our third year project. It is a job seeking application where an employer can post a job listing using the web application and a job seeker can apply to a listing with the mobile app. The goal of appli was to connect a job seeker to an employer and create a bridge for new opportunities. We implemented the “Tinder” mechanic on the mobile app, where a user is displayed a job listing with all the necessary information they need to know if they’re interested in the job. The user would then swipe the listing right or left to save it to their profile or dismiss it respectively. The next listing would then be displayed to the user and the process repeats.

SocialPath

Web App, Angular

In order to help me learn Angular I have decided to make an application that will link into Facebook. The app will asynchronously retrieve data and updates from the user’s Facebook profile and display it on screen. I’m hoping to create a dashboard type of interface with posts, updates, messages and notifications all having their own sections. With further developments and with greater understanding of Angular in the future I may add other social media’s to this dashboard such as twitter or Instagram. A user could be able to view all updates on the one dashbord, or separate them out to only see twitter for example.
